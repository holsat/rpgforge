# Tauri places generated icons here — keep this folder.
# Run `npm run tauri icon path/to/source.png` once you have a logo PNG (1024x1024).
# That command writes 32x32.png, 128x128.png, 128x128@2x.png, icon.icns, icon.ico.
#
# Until then, Tauri will fail `tauri build` because icons are required for bundling.
# `tauri dev` works without them.
